# Hierarchical ORGaNICs Model

This repository implements a hierarchical version of the ORGaNICs (Oscillatory Recurrent Gated Neural Integrator Circuits) model, focusing on Inter-areal communication between V1 and V4 areas of the brain. 

## Project Structure  
 
├── Analysis/ # Analysis scripts for model outputs  
├── Models/ # Core model implementations  
├── Utils/ # Utility functions and helpers  
├── Job_scripts/ # Cluster job submission scripts  
├── Results/ # Model results and outputs (not tracked)
